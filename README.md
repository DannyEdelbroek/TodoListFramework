# Todo App

Dit is een overzicht van de UI voor mijn Todo-app. Het laat zien hoe de dashboardpagina eruit zal zien, inclusief de taken voor de week, categorieën en voortgang van projecten.

![Todo App Dashboard](./a4567f39-efe9-4dfc-99ad-307326f904cc.png)

## Features

- [] Taken organiseren per dag, week of alles
- [] Categorieën voor school, werk, side-projecten en privé
- [] Volg de voortgang van projecten en reflectie per week
- [] Voeg nieuwe taken toe via de “Nieuwe taak” knop
- [] Duidelijke statuslabels voor taken: open, afgerond, etc.
- [] RoleSysteem maken.
- [] Statastieken pagina maken.


## Doel

Het doel van deze app is om mensen een overzicht te geven van hun taken en voortgang, met een duidelijke en overzichtelijke UI die eenvoudig te gebruiken is.
